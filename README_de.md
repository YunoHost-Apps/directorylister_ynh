## Übersicht

Directory Lister is the easiest way to expose the contents of any web-accessible folder for browsing and sharing.

### Features

- Simple installation
- Light and dark themes
- Custom sort ordering
- File search
- File hashes
- Readme rendering
- Zip downloads
- Multi-lingual


**Ausgelieferte Version:** 5.0.4~ynh1
